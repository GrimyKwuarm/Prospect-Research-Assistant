# Deployment

## Status

No deployment target exists yet.

## Current Position

This project should remain local and documentation-first until the evidence model and first prototype workflow are agreed.

## Decisions Required Before Deployment

Before any deployed system is built, decide:

- whether the tool stores source material or generated profiles,
- whether authentication is required,
- who the intended users are,
- how sensitive and donor-level data will be protected,
- whether outputs are logged,
- what retention and deletion rules apply,
- whether any external APIs are used.

## Near-Term Recommendation

Start with a local Codex-native workflow or small local prototype before considering deployment.
